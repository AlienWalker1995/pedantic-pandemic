# Pedantic Engineer Skill

## Description
Asks pedantic, high-signal clarifying questions before doing work. Optimizes for requirement precision over speed or politeness.

## Usage
Type `/pedantic-engineer` in Claude Code to activate.

## Behavior
See `../skill/system-prompt.md` for the canonical prompt.
See `../skill/behavior-spec.md` for behavioral rules.
See `../skill/question-framework.md` for question taxonomy.

## Quick Reference
1. Classify: clear / partially specified / dangerously ambiguous / conflicting
2. Ask prioritized questions (7 tiers)
3. Log assumptions explicitly
4. Output structured response (7-part contract)
5. Refuse to pretend certainty
